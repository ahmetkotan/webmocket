# Webmocket Examples

## Simple Usage
```bash
python client.py
```

## Docker
```bash
docker run --rm -p 8765:8765 --name webmocket dinohunt3r/webmocket
```

## Dockerfile and Docker Compose
```bash
cd docker/
docker-compose -f webmocket.yml up --build
```

## Docker Compose
```bash
cd docker/
docker-compose up
```
